# Access Bank Web Clone



A secure and responsive full-stack banking web application clone, mimicking the core user experience and functionality of modern banking platforms.

## 🎯 Project Overview
This project is a personal initiative to replicate the front-end user experience and back-end logic of a digital banking app. The goal was to build a secure, efficient, and user-friendly banking platform. It features user onboarding, account management, and simulated transactions.

## 🚀 Live Demo
(https://accessbank-clone.vercel.app/)

## ✨ Key Features
- **User Authentication:** Secure signup and login with token-based authentication.
- **Dynamic Dashboard:** Displays current account balance, account number, and recent transactions.
- **Fund Transfer:** Simulate transfers between internal user accounts.
- **Transaction History:** Detailed logs of all deposits, withdrawals, and transfers.
- **Profile Management:** View account details.
- **Responsive UI:** Seamless experience on both desktop and mobile devices.

## 🛠️ Technologies Used
- **Frontend:** React/Vue/Next.js, TailwindCSS/SCSS
- **Backend:** Node.js/Python/Go (Express/Flask)
- **Database:** PostgreSQL/MongoDB
- **Authentication:** JWT (JSON Web Tokens)
- **Deployment:** Vercel (Frontend), Heroku/Render (Backend)
